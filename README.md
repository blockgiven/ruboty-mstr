# Ruboty::Mstr

ruboty plugin to get delicious food pictures.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-mstr'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-mstr
